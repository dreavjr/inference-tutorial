# Inference: from classical to Bayesian

*An interactive primer on reasoning and acting over uncertain data.*

Set the hidden truth, draw samples, and watch the statistical machinery respond — then try it in Python yourself.

**Read it online: https://dreavjr.github.io/inference-tutorial/**

## Contents

1. How classical tests work — the t statistic, its null distribution, and what a p-value measures.
2. The test vs. the world — power and significance, and what they don't tell you.
3. The Bayesian approach — prior × likelihood ~ posterior, then integrate away the rest.
4. Making a decision — turning a posterior into a decision via expected utility.
5. A Bayesian t-test? — the same results, a radically different interpretation.
- Do it yourself — a Beta–Binomial model in Python, with a companion notebook.
